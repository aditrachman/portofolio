## Step 2 â€“ Upload

Buka terminal / cmd di folder yang akan di upload:
atau kalau kamu menggunakan windows 11 bisa klik kanan di folder yang akan di upload lalu klik kanan.


lalu tinggal kalian ketikan ini di terminal

```
git init
```

selanjutnya

```
git add .
```

dan selanjutnya kalian isikan pesan untuk commit pertama

```
git commit -m "ini upload file pertama ku"
```

lalu pilih branch atau cabang di mana kalian mau mengupload nya
biasa nya github otamatis menyarankan ke cabang main

```
git branch -M main
```

lalu remote repository yang sudah kalian buat tadi dengan cara

```
git remote add origin https://github.com/username_kalian/nama_repo_kalian.git
```

langkah selanjutnya sekaligus langkah terakhir adalah mengupload

```
git push -u origin main
```
